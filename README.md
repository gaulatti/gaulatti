# Gaulatti (React)

Personal website built with React Router and Tailwind CSS.

## Features

- SPA mode (`ssr: false`)
- Static production output in `build/client`
- TypeScript + Tailwind CSS

## Getting Started

### Installation

Install the dependencies:

```bash
npm install
```

### Development

Start the development server with HMR:

```bash
npm run dev
```

Your application will be available at `http://localhost:5173`.

## Building for Production

Create a production build:

```bash
npm run build
```

## Deployment

Deploy the contents of `build/client` after running:

```
npm run build
```
